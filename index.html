Digenpy by XayOn

Dictionary generators for default wireless installs

Digenpy was originally born to be integrated with airoscript-ng but became soon a greater project.
Right now, it's able to generate password dictionaries for more than 10 different kinds of routers and 3 different companies, even the famous WPA Comtrend default passwords.
It works on android, windows, and linux, and has a GTK graphical interface, besides from the standard old CLI interface.

Dependencies

python, pygtk

Install

For a normal installation with an uncompressed tarball/zip or a git clone, execute as root: python setup.py install

Install on windows

To install on windows you can try the following installer: Installer for windows

Install on android

Try (experimental) this APK: Installer for droid

Install on debian

There's also available a debian package, got it from the debian logo at the top of the page, and execute: dpkg -i package.deb
